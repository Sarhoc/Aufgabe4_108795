## Introduction to Pandas DataFrames - Replicating an article in Markdown
1. Notebook file (`Pandas.ipynb`)
   The core element - the Jupyter Notebook containing the text and code.
2. HTML file (`Pandas.html`)
   An additional file that allows the notebook to be viewed without Jupyter.
3. Image (`Pandas.webp`)
   The image embedded in the original article and used in the notebook. It needs to be stored in the project directory so that it can be displayed.
